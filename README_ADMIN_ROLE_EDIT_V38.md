# v38 管理員開通帳號方式

這版依照「玩家註冊後，管理員直接在 Supabase 後台改等級」的方式整理。

## 玩家註冊後資料在哪裡？

Supabase → Table Editor → `app_users`

玩家註冊後會自動出現在這張表。

## 管理員要改哪個欄位？

只要改 `role` 欄位。

可用角色：

- `體育萌新`
- `一般會員`
- `VIP會員`
- `體育大神`

## 開通規則

### 剛註冊
預設是：

`體育萌新`

### 找管理員開通
把 `role` 改成：

`一般會員`

### 加入專屬會員群
把 `role` 改成：

`VIP會員`

### 特殊活動或人工派發稱號
把 `role` 改成：

`體育大神`

## 需要看的欄位

- `account`：玩家帳號
- `password`：玩家密碼
- `nickname`：玩家暱稱
- `line_id`：玩家 Line ID
- `role`：目前會員等級
- `month_recommended`：本月預測場數
- `month_wins`：本月命中場數
- `is_online`：是否在線

## 體育大神

每月可領當月限定好禮，每月好禮不一樣，歡迎多多參與。

## 升級規則

1. 玩家權限為【VIP會員】
2. 當月預測滿 50 場
3. 準確率為 70% 以上
